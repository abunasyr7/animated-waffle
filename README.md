# Конвертер азбуки Морзе

Это финальная задача 6-го спринта курса «Go-разработчик с нуля». В рамках проекта предстоит реализовать веб-приложение, которое преобразует текст в код Морзе и обратно.

## Что нужно сделать

Выполнив этот проект, вы научитесь:

1. Создавать HTTP-сервер.
2. Парсить форму HTML.
3. Создавать и регистрировать хендлеры в маршрутизаторе.
4. Подгружать файлы из HTML-формы.
5. Автоматически определять содержимое файла (текст или код Морзе).
6. Использовать методы из пакета, который для вас написан, чтобы конвертировать текст в код Морзе или наоборот.

## О проекте

Приложение должно принимать данные от пользователя через HTML-форму, обрабатывать файл или текстовое поле и возвращать результат конвертации. Важно, чтобы приложение автоматически определяло формат входных данных и выполняло нужное преобразование.

## Структура

- `cmd/` — точка входа приложения.
- `internal/` — реализация сервера, хендлеров и бизнес-логики.
- `pkg/morse/` — пакет для работы с кодом Морзе.

## Цель

Задача помогает закрепить навыки разработки веб-сервиса на Go и работу с формами, файлами и маршрутизацией в рамках курса.
